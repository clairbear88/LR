Stage 1: HTML and CSS

Basics of HTML

HTML Documents:

Made up of text content, markup, references to other docs, and links to other pages.

Markup:
Elements: <tag attribute="example">text content<tag>
Tags:
  • Links: "a" with attribute href, non-void, inline
  • Images: "img", attr(s)=src/alt, void, inline
  • Return: "br", void, inline
  • Paragraph: "p", non-void, block
  • Contents Style Containers: "span", non-void, inline; "div", non-void, block

Block v.s. Inline: Block elements separates contents from the rest of contents in HTML doc, Inline leaves no separation and is inline with the rest of your content.
Void vs. Non-Void: Void tags do not require a closing tag, Non-Void do.

Misc:
  • Computers are stupid...
  • HTML stands for Hypertext Markup Language

HTML Structured Documents

Using Chrome's Developer Tools to Inspect HTML Elements:

Everything is a box. Block Elements can be used to form an invisible box around segments of content.

Tree-Structure of HTML:

To give HTML Documents structure, elements can be placed into a hierarchy and segments of content can be created by nesting elements inside of elements.

Document Object Model(DOM): is a document structure standard that defines the way programs will interact with a document type.

Adding CSS Style and HTML Structure

Prepping your HTML Doc

Adding <!DOCTYPE html>,<html>, and <head> tags.
Adding a link to CSS stylesheet: <link rel="stylesheet" href="docname.css">

Cascading Style Sheets(CSS):
  • Uses selectors to find class or tag of an element and uses a declaration rule made up of a property and value.
  • Order Matters and if parent property values overlap the more specific child value is applied to the element.
  • Some Browsers use slightly different styles.
  • Style can be applied a separate document, inline in an element, or placed in <style> tags.
  • Comments are opened with /* and closed with */

Stage 2: Python

Strings and Variables

Variables:

Variables like in math are symbols or strings that are assigned a value. Anytime the variable is used a computer substitutes the assigned value for the variable that was used:

x=1
y=2
print x+y
returned value: 3

Strings:

Text contained between single quotes: 'I am a string'
Each position of a string can be referenced by an index number using brackets: 'I am a string'[0:3] equals 'I am'
Find function or procedure can be used to find the position of a substring of a string: 'I am a string'.find('string')

Putting it together:

stringy='I am a string'
stringier=stringy[0:3]
print stringier.find('am')
returns 2
